💰 Expense Tracker

An advanced expense tracking web app built with Python and Streamlit.

Features:
- Add expenses with category and description
- View all expenses with total amount
- Filter by month and category
- Pie chart of spending by category
- Budget limit with alerts
- Data saved permanently in CSV file

Tech Stack:
Python | OOP | CSV | Streamlit | Pandas | Matplotlib

How to Run:
pip install streamlit pandas matplotlib
streamlit run app.py
